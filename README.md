# Proyecto Final - Backend de Adopciones

Incluye documentación Swagger del módulo Users, tests funcionales del router de adopciones y Dockerización de la aplicación.

## Tecnologías utilizadas

- Node.js
- Express
- MongoDB
- Swagger
- Docker
- Mocha
- Chai
- Supertest
